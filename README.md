# Interview Prep

Structured prep for DSA, SQL, System Design, and CS Fundamentals — powered by Claude Code.

## Get started

```bash
git clone git@github.com:onkarnik-personal/InterviewPrep.git
cd InterviewPrep
npm install
git checkout -b <your-github-username>
git push -u origin <your-github-username>
```

Work only on your branch. Never push directly to `main`.

## Run a solution

```bash
npm run solve problems/arrays-strings/twoSum.ts
```

## Structure

```
problems/        # one file per problem, organised by pattern
docs/plan.md     # 3-month roadmap with checkboxes
docs/progress.md # daily log
docs/cs-fundamentals/
docs/system-design/
docs/patterns/
```

Read `CLAUDE.md` for the full session workflow.
